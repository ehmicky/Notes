Voici des options communement acceptes. La notation abregee est UNIX-like, la notation avec double-dash est GNU-like :

-a      --all           All: show all information or operate on all arguments.
-f      --file          File : filename follows.
-f      --force         Force : force overwrite of target file(s).
-h      --help          Help : gives help message and exit
-l      --list          List: list files or arguments without taking action.
-o                      Output filename
-q      --quiet         Quiet: suppress stdout.
-r
-R      --recursive     Recursive: Operate recursively (down directory tree).
-v      --verbose       Verbose: output additional information
-V      --version       Show program version and exit.
-z      --compress      Compress: apply compression (usually gzip).
