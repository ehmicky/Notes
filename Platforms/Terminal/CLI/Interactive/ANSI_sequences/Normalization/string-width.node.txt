STRING-WIDTH(STR[, OPTS])->NUM    #Gets STR width:
                                  #  - ignoring:
                                  #     - C0|C1 control characters
                                  #     - Unicode combining characters (U-0300 to U-0-036f) like accents
                                  #     - ANSI sequences (using STRIP-ANSI, see its doc)
                                  #  - counting as 2:
                                  #     - fullwidth Unicode chars (using IS-FULLWIDTH-CODE-POINT, see its doc)
                                  #     - emoji (using EMOJI-REGEX, see its doc)
                                  #  - handles characters >=U+10000
                                  #OPTS:
                                  #  - ambiguousIsNarrow BOOL:
                                  #     - some characters have width of 2 in East Asian legacy encodings, but 1 otherwise
                                  #     - if true (def), width is 1. If false, it is 2
