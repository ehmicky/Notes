Crypto :
  - confidentialité :
    - communication secrète :
      - militaire
      - diplomatique
    - transaction financière :
      - e-commerce
      - CB
    - voting system
    - secure cloud computing
    - crypto asym. + tamper-resistance pour permettre une opération que par l'intermédiaire d'un système donné :
      - e-cash
      - DRM (DVD, BluRay)
  - intégrité :
    - pour un produit donné :
      - OS
      - torrents
  - authentication :
    - non-phishing 
      - notamment dans transaction financière
    - autorisation des composants d'un système :
      - app store
      - jeu vidéo sur une console
      - trusted computing
  - non-repudiation :
    - digital signatures
