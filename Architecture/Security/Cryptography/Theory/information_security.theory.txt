Computer security :
  - comme information security, mais limité aux échanges informatiques
  - information security peut aussi concerné échanges papiers, etc.

Information assurance :
  - risk management pour la sécurité informatique

Au-delà de la crypto security :
  - EMSEC/TEMPEST :
    - émission électro-magnétique
  - tamper resitance :
    - ouverture physique d'un appareil physique matériel
  - tamper evidence : 
    - détection d'une ouverture physique
